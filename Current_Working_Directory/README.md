# PDF Diff Project

Welcome to the repository!

## ⚠️ Important Note for Collaborators

The primary active development is happening inside the **`Current_Working_Directory`** folder. 

Specifically, please focus all your efforts, new features, and contributions entirely on these two subdirectories:
* **`Kv_Image`**
* **`Kv_Text`**

These two folders contain the current working versions of the project.

### Other Directories and Files

You will likely notice other files and directories throughout this repository. Please treat all other locations as experimental tests, prototypes, and older iterations that were used as stepping stones to arrive at this current state. 

To keep our workflow clean and organized, please ensure all your work happens exclusively within the `Kv_Image` and `Kv_Text` directories. Happy coding!
