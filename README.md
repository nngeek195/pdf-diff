# PDF Diff Project

Welcome to the PDF Diff Project repository. This project focuses on comparing and differentiating PDF documents.

## 📁 Repository Structure & Active Development

To maintain a clean and organized workflow, please note that **active development is restricted to specific directories.** 

The core of the current project resides in the following path:
**`Current_Working_Directory/`**

Within this directory, there are two primary modules where all new features, bug fixes, and contributions must be directed:
* **`Kv_Image`**: Dedicated to image-based PDF comparison and processing.
* **`Kv_Text`**: Dedicated to text-based PDF extraction and differentiation logic.

### ⚠️ Important Guidelines for Collaborators

1. **Focus Area:** Please ensure that all your commits and modifications are made exclusively within the `Current_Working_Directory/Kv_Image` and `Current_Working_Directory/Kv_Text` folders. 
2. **Legacy & Experimental Code:** You will find various other files and directories at the root level and outside of the `Current_Working_Directory`. Please treat these as experimental tests, prototypes, and older iterations. They serve as a historical reference for how the current architecture was achieved but are **not** part of the active production codebase. 

Please do not build upon or modify these legacy files.

## 🚀 Getting Started

When contributing, please navigate directly to either `Kv_Image` or `Kv_Text` depending on your assigned task. If applicable, review any sub-documentation within those specific directories for environment setup and execution instructions.

Thank you for your contributions and for adhering to the repository guidelines!
